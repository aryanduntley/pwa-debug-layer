- Wire: MCP -> IPC -> NMH -> SW -> CS -> page-world dispatcher (existing path).

  call new Function('return ('+expression+')')() with try/catch, optional

  Page-world eval is materially different from chrome-devtools-mcp evaluate —

  per note 51.
